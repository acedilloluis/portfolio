    Portfolio

    Here are some of my projects.

    Todo App

    A todo app built using React and a mobile-first workflow. It allows the user to create, edit and delete todos. Once created the user can filter between all, active and completed todos. The app dynamically tracks the amount of todos. It also allows the user to clear the completed todos. The app is entirely keyboard accessible. View the live site and the Github repository .

    Sunnyside Agency Landing Page

    A responsive landing page built with no frameworks using CSS grid and a mobile-first workflow. View the live site and the Github repository.

    Crowdfunding Product Page

    A responsive crowdfunding product page built with no frameworks using a mobile-first workflow. The page will dynamically track the number of items left and the total amount of money raised. The page also opens a modal window to allow the user to select an item and input a pledge amount. Once this is done, it opens another modal window thanking the user. View the live site and the Github repository.

    About

    My name is Luis Cedillo. I'm a self-taught front-end developer. I started working with HTML and CSS in 2015 when I took a web development course in high school. For a little while after the course I continued to build small websites, but did not really delve any deeper into front-end development. My journey started up again almost 7 years later when I decided to dive head-first into the field.

    Now I'm focusing on learning mobile-first, responsive web-design as well as JavaScript. My goal is to create fast, easy to use and accessible websites and web apps. To this end, I'm comfortable working with HTML5, CSS 3 and JavaScript/React.
