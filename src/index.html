    About

    My name is Luis Cedillo. I'm a self-taught front-end developer. I started working with HTML and CSS in 2015 when I took a web development course in high school. For a little while after the course I continued to build small websites, but did not really delve any deeper into front-end development. My journey started up again almost 7 years later when I decided to dive head-first into the field.

    Now I'm focusing on learning mobile-first, responsive web-design as well as JavaScript. My goal is to create fast, easy to use and accessible websites and web apps. To this end, I'm comfortable working with HTML5, CSS 3 and JavaScript/React.
